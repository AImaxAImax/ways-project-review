# Visual Quality Target v7 — Premium Cut-Out Diorama

## Goal
Keep the v6 cut-out / paper diorama idea, but upgrade the output quality so it feels intentional and premium instead of simple placeholder vector art.

## Quality upgrades
- Larger canvas: render at 2160×3840, downsample to 1080×1920 for cleaner edges.
- Real paper feel: subtle fiber grain, uneven handmade paper, soft color variation.
- Layered depth: foreground/midground/background paper layers with cast shadows.
- Better lighting: warm museum spotlight / vignette / rim glow.
- Better composition: strong subject silhouettes and negative space for captions.
- More polished props: apple/squirrel/acorn, fossil slab, timeline shelves, aquarium silhouettes.
- Unified palette: teal ocean, ochre paper land, amber highlights, cream fossil paper, dark navy shadows.

## What not to change
- Keep “say dog see dog.” Each line still needs its own literal visual payload.
- Keep consistent art language across all shots.
- Do not return to generic AI shark portraits.

## Pipeline recommendation
Use v7 controlled cut-out frames as the new high-quality layout/keyframe base. If we want even more polish later, use these as img2img/control references, not pure text-to-image prompts.
