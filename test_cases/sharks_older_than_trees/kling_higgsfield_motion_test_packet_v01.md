# Kling / Higgsfield Motion Test Packet v01

Purpose: decide whether paid AI video animation is worth it versus static Ken Burns for the shark short.

## Verdict to test
The Ken Burns probe proves pacing/shot order, but it does **not** prove final motion quality. The final should have real subject/environment motion: water, shark body motion, camera drift, particles, light rays, foreground parallax, aquarium glass, etc. Keep motion restrained and cinematic — no rubbery morphing, no AI wobble.

## Global motion style
- Vertical 9:16, 5 seconds per clip if available.
- Use image-to-video from the still frame as first frame/reference.
- Premium natural-history documentary style.
- Subtle realistic movement, not a hallucinated action scene.
- Preserve the exact composition and main subjects from the reference still.
- Camera: slow dolly/push/pan only.
- Atmosphere: water particles, dust, mist, light rays, gentle environmental movement.
- Avoid: morphing anatomy, extra sharks unless requested, scary attack energy, gore, rapid cuts, fake text, distorted captions, rubbery wobble, melting objects, changing species.

## Test clips to generate first
Do **not** animate every shot yet. Test these 4 clips first. If 3/4 are clearly better than the Ken Burns probe, Higgsfield/Kling is worth it for this video.

## Buy / no-buy decision rule
Buy/keep Higgsfield/Kling only if the test clips produce **actual believable motion** that is obviously more compelling than the Ken Burns probe:

- 3 of 4 test clips must preserve composition and not morph.
- At least 2 clips must show meaningful subject/environment motion, not just zoom.
- No clip should introduce fatal story errors like forests in the “before forests” beat.
- If only the aquarium/final shark shots work, use Kling selectively for those hero shots and keep still-motion for the rest.
