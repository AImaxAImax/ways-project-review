# Voice casting notes

## Problem with rough VO
The Edge TTS proxy voice is useful for timing only. It is not publishable. It sounds synthetic, flat, and generic.

User rejected the current voice options. Treat the voice lane as blocked for final concept validation until a higher-quality humanlike narrator is auditioned. Do not generate more Edge voice samples as if they are contenders; use them only for timing scratch tracks.

## What we need instead
A voice that feels like:
- warm
- curious
- trustworthy
- slightly amused
- not babyish
- not radio-announcer polished
- not TikTok hype voice
- not “educational AI channel”

## Casting direction
Think: older sibling / cool science teacher / bedtime-adjacent narrator.

Delivery notes:
- Start the first line like a secret: “Sharks were here before trees.”
- Let “Seriously.” breathe.
- Smile slightly on the squirrels line.
- Slow down on “four hundred million years ago.”
- End with quiet awe, not hype: “a survivor from a world before forests.”

## ElevenLabs prompt/settings direction
Voice type:
- adult narrator
- warm mid-range voice
- natural conversational pacing
- gentle confidence
- subtle wonder

Avoid:
- child voice
- character/cartoon voice for main narration
- overexcited YouTuber cadence
- spooky documentary bass voice
- synthetic newsreader tone

## Test reads to generate
1. Warm male narrator
2. Warm female narrator
3. Slightly younger energetic narrator, but still parent-safe

See `voice_audition_packet.md` for the current casting packet and acceptance test.

## Local/free voice role
Use local or free voices only for rough timing. Final test should use a better TTS/provider if we want to honestly judge the channel concept.
