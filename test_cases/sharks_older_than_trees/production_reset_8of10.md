# Production Reset — From 5/10 POC to 8/10+ Short

## Honest assessment
The v6/v7 cut-out experiments proved the *direction* but not the final production value. They are useful POC/layout references, not publishable final art.

Current issue: the controlled paper-cut output is too handmade/simple in the wrong way. It solves consistency and “say dog, see dog,” but it does not yet feel like a premium YouTube Short package.

## New production bar
Target: **8/10+ publishable test**, not rough proof.

The final visual package needs:
- premium editorial art direction, not placeholder shapes
- consistent brandable art style across frames
- distinct settings/props per narration beat
- strong silhouettes readable on mobile
- cinematic lighting/depth/material texture
- curated best-of generations, not first acceptable outputs
- proper compositing/motion/camera after stills are selected

## Production strategy shift
Stop trying to make the simple Python paper-cut renderer do final-quality work. Use it only as a **layout/reference lock**.

New pipeline:
1. **Art director style-frame challengers** — generate a few premium single-frame looks first.
2. Pick the best style language.
3. Create locked shot prompts + visual references per narration line.
4. Generate 6–12 candidates per shot using the selected premium style.
5. Curate one frame per shot against a checklist.
6. Assemble with camera moves, captions, SFX/music, and voice.

## Candidate premium styles to test
### A. Premium tactile paper diorama
High-end layered paper sculpture photographed in a miniature studio. Real paper fibers, cast shadows, macro lens, museum lighting. Still kid/family friendly but much more polished than v7.

### B. Natural-history storybook 3D
Stylized 3D clay/paper hybrid like a premium children’s science book trailer. More depth, richer materials, better characters/props.

### C. Editorial museum infographic diorama
A refined National Geographic Kids / museum exhibit hybrid: diorama scenes, fossil displays, timelines, warm gallery lighting, clean educational composition.

## Rejection criteria for next pass
Reject if:
- it looks like placeholder vector/PIL art
- it looks like generic AI fantasy art
- it repeats the same shark/ocean scene
- it lacks the literal object/setting for the narration beat
- it has AI text artifacts
- it would not make someone stop scrolling on mobile

## Current role of prior files
- v6/v7 remain reference/layout documents.
- They are not the visual ceiling.
- The next pass should use stronger image generation/art direction, not just more local procedural drawing.
