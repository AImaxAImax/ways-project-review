# Visual Reassessment v21 — Museum is a tool, not the whole video

## Decision
User and director agree: the current path drifted too heavily toward museum/stage shots. Museum language is useful for proof/evidence, but it should not become the whole visual identity.

## New operating rule
**Same premium quality, different visual worlds.**

Consistency comes from palette, lighting, tactile texture, macro depth, family-safe natural-history tone, and clean mobile composition. Variety comes from different settings and visual nouns.

## Museum usage limit
Use museum/exhibit/proof language for roughly **20–30%** of the short:
- Shot 2 “Seriously” fossil proof
- optional accents in timeline/comparison, if needed

Avoid museum/stage language for the rest:
- Shot 1 should be immersive ancient ocean + barren land.
- Shot 3 should be a literal comparison gag, not another fossil table unless control requires it.
- Shot 5 should be primitive land/early forest arrival.
- Shot 6 should be modern aquarium/viewer POV.
- Shot 7 should be final ancient-ocean awe.
