# Plate prompt packet — Saturn hexagon storm

Global negative prompt for every generated/local plate:
`no text, no captions, no labels, no numbers, no logos, no NASA insignia, no user interface, no diagrams with words, no watermark, no fake spacecraft dashboard, no lightning apocalypse, no explosions, no faces, no people, no extra planets, no Earth visible unless specifically requested, no unreadable micro-text`

Global style target:
`premium cinematic science documentary, Cassini-inspired Saturn north pole, warm gold and amber cloud bands, deep black space, clean silhouette, high phone-size readability, caption-safe negative space in the center-upper third, realistic gas giant atmosphere, subtle volumetric light, no clutter`

## Shot 01 — Hook: impossible shape
VO: "Saturn has a storm system shaped like a hexagon."
Duration: 4.0s
Preferred lane: NASA-source crop + local motion graphic emphasis.
Source reference: PIA21052 or PIA14646.
Prompt if local still needed: `vertical 9:16 cinematic view looking down on Saturn's north pole, a clear six-sided hexagon-shaped cloud jet stream visible in golden atmosphere, black space around the planet, dramatic but realistic Cassini science image feel, clean center composition, no text, no labels`
Motion note: start close on abstract clouds, quick pullback reveals the six-sided pattern by 1.5s.
Gate 2 pass bar: hexagon reads instantly at phone/contact-sheet scale.

## Shot 02 — Not a circle
VO: "Not a circle. Not a swirl. A six-sided pattern around its north pole."
Duration: 5.5s
Preferred lane: motion graphic prototype over NASA source; final can be source crop with a brief non-text hexagon outline accent.
Prompt if local still needed: `overhead Saturn north pole, hexagonal cloud boundary crisp and readable, polar clouds swirling inside, realistic amber and cream atmosphere, clean dark-space border, documentary realism, no text or UI`
Motion note: cut/punch-in on each negation; outline appears briefly then fades.
Gate 2 pass bar: no fake labels; outline cannot feel like a classroom diagram.

## Shot 03 — Cassini proof
VO: "NASA's Cassini spacecraft saw it from above, and each side is about as wide as Earth."
Duration: 7.0s
Preferred lane: NASA PIA21052 primary proof crop.
Prompt if local still needed: `Cassini-inspired overhead scientific image of Saturn's north polar hexagon, huge scale, cloud bands and polar vortex visible, cinematic realistic processing, vertical crop with empty space for captions, no text, no scale bars, no Earth`
Motion note: slow orbital drift; optional tiny non-text Earth-scale circle only if later approved, otherwise rely on captions/VO.
Gate 2 pass bar: fact proof must not invent a visible Earth or fake measuring graphic unless explicitly approved.

## Shot 04 — Storms inside
VO: "Inside the hexagon are smaller storms, wrapped around a huge vortex at the pole."
Duration: 7.0s
Preferred lane: NASA PIA17122 / PIA20507 source crop.
Prompt if local still needed: `close overhead view inside Saturn's north polar hexagon, many smaller cloud storms and a dark central polar vortex, realistic gas giant cloud texture, amber monochrome Cassini palette, elegant science documentary look, no text, no labels`
Motion note: gentle inward push toward vortex; no hurricane-on-Earth visuals.
Gate 2 pass bar: vortex and nested storms visible without becoming scary or messy.

## Shot 05 — Jet stream/fence metaphor
VO: "Scientists describe the shape as a jet stream, with winds tracing the edges like an invisible fence."
Duration: 8.0s
Preferred lane: local motion graphic over NASA cloud texture.
Prompt if local still needed: `Saturn north pole hexagon cloud jet stream, six straight-ish wind bands forming a stable geometric boundary, cinematic cloud flow, realistic planetary atmosphere, elegant minimal composition, no text, no arrows, no UI`
Motion note: subtle cloud flow along edges; if using graphics, use thin glowing wind streaks with no arrows/text.
Gate 2 pass bar: metaphor reads visually but does not imply a solid wall.

## Shot 06 — Final crown loop
VO: "So Saturn doesn't just have rings. It has a planet-sized six-sided storm crown."
Duration: 7.0s
Preferred lane: PIA14646 rings + north pole if readable, or cinematic Saturn-wide local plate based on NASA refs.
Prompt if local still needed: `majestic vertical view of Saturn with rings partially visible and the north polar hexagon glowing subtly on top like a storm crown, realistic Cassini-inspired gas giant, black space, warm gold clouds, awe-filled science documentary, no text, no logos`
Motion note: pull back to Saturn/rings, then loop visually toward hexagon shape.
Gate 2 pass bar: poetic "crown" read without adding literal crown objects.
