# Production packet — First-10 item 080 — Saturn hexagon storm

## Package status
Gate 1 packet only. No credits spent. Public-domain/NASA-first source plan prepared; rendering remains held until Gate 1 script/fact approval and Gate 2 plate approval.

## Core creative thesis
Saturn is already familiar because of its rings, but the stronger short-form surprise is that its north pole has a persistent six-sided storm/jet-stream pattern. The video should reveal the hexagon immediately, then prove it with Cassini/NASA imagery and a clean cinematic interpretation.

## Target format
- 35-42s vertical Short, 1080x1920 master, 720x1280 Discord preview.
- WAYS caption system: large centered white captions, yellow active/key word, thick black outline; no bottom block.
- Zero non-caption text in plates, source crops, generated stills, or motion graphics.
- Visual reset every 2-3 seconds.
