# Research notes — Saturn hexagon storm

## Verified source snippets gathered 2026-06-06 UTC
- NASA Images API result PIA21052, "Over Saturn Turbulent North": metadata says the Cassini view shows part of the giant, hexagon-shaped jet stream around Saturn's north pole. Each side of the hexagon is about as wide as Earth. A circular storm lies at the center, at the pole. Image taken Dec. 3, 2016; distance about 240,000 miles / 390,000 km; image scale 14 miles / 23 km per pixel.
- NASA Images API result PIA17122, "Stormy North": metadata says the area within Saturn's north polar hexagon contains myriad storms of various sizes, including the remarkable/imposing vortex over the north pole.
- NASA Images API result PIA14646, "Hexagon and Rings": metadata says Saturn's north polar hexagon is sunlit, smaller storms dot the north polar region, and rings appear in the background.
- NASA Images API result PIA20507, "Saturn Watercolor Swirls": metadata says Saturn's northern polar region is dominated by the famous hexagon shape, circumscribing the northern polar vortex.
- NASA image/media guideline page metadata says NASA content/images/audio/video/media files are generally not subject to U.S. copyright for non-commercial use. Still avoid implying NASA endorsement and avoid NASA marks/logos.

## Production interpretation
Use NASA/Cassini imagery as the accuracy anchor. The cinematic layer should enhance readability for mobile, not invent features. A non-text hexagon outline or wind streaks can help the audience see the geometry if Gate 2 approves it.
